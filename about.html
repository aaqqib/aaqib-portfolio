How this site was made

This portfolio is a completely custom-built static site designed for maximum speed and SEO performance. No heavy frameworks, no bloated templates.

  • HTML/CSS/JS: Vanilla web technologies for zero bloat.
  • Feather Icons: For clean, lightweight vector icons.
  • Netlify CMS: A headless CMS to easily manage the blog and writing sections without needing a database.
  • Vercel: For lightning-fast global edge network hosting.

The design is inspired by modern, minimalist interfaces, utilizing CSS variables for a seamless dark mode transition.
